如何在华为云上购买ECS及以镜像的方式部署华为云欧拉操作系统 (HCE OS)

对于公有云厂商的 ECS 创建，接触过 openstack 私有云的小伙伴想来很熟悉，基本上是同一套流程

  • 创建子网和安全组
  • 创建或者选择 flavor(硬件资源配置)
  • 创建或者选择镜像
  • 创建实例，包括预装配的一些操作
  • 启动实例，绑定浮动 IP

今天和小伙伴们分享一下华为云上 ECS 的创建，镜像使用华为云的开源系统欧拉。

创建虚拟私有云

点击右上角控制台，选择Region为 离当前业务最近的区域。鼠标放到左侧服务列表符号，点击网络部分第一个服务：虚拟私有云 VPC。

点击右上角 创建虚拟私有云。关于什么是 虚拟私有云 VPC ，可以简单理解为独属于自己 弹性云服务器 的一个局域网，可以暴露一个 公网 IP 供你远程连接

输入参数如下：

  • VPC 名称：vpc-hce
  • 子网名称：subnet-hce

其他选项保持默认，点击右下角“立即创建”。在公有云上面创建 VCP 一般会默认自动开启 DHCP，一个 VPN 子网可以供多个 ECS 使用，需要注意可网段可分配 IP 的个数。

创建完成后可在列表中查看到此 VPC。点击左侧网络控制台中的访问控制，点击安全组。点击右上角“创建安全组”。

安全组可以理解为 可视化的防火墙配置，对出站入站的端口做限制

一般的会提供一些默认的模板，比如 SSH，Web，也可以定义自己的规则

购买 ECS

在左侧服务列表中选择计算类服务第一项：弹性云服务器ECS。击右上角“购买弹性云服务器”。

ECS 具体配置参数如下：

  • 计费模式：按需计费
  • 区域：华东-北京四
  • 可用区：随机分配

这里的计费模式根据实际情况选择，区域选择最近的区域，然后是选择资源模板，即购买的 ECS 需要什么硬件配置，默认情况下会提供一些选择模板，这在 openstack 中被称为 flavor 需要注意， 如果系统镜像对配置有要求，那么较小的 flavor 可能无法启动实例

之后需要选择使用的镜像，这里我们使用华为欧拉系统

配置完成后点击“下一步：网络配置”，进入网络配置，配置网络参数如下：VCP 和子网选择我们之前创建好的，同事安全组也选择之前创建的

  • 网络：vpc-hce | subnet-hce | 自动分配 IP 地址
  • 安全组 ：sg-hce
  • 弹性公网 IP：现在购买
  • 线路：全动态 BGP
  • 公网带宽：按带宽计费

配置完成后，点击“下一步：高级配置”，配置 ECS 高级配置

这里云备份要选择暂不购买，不选择会默认开启云备份，自动扣款

确认配置无误后勾选“我已阅读并同意镜像声明”，点击“立即购买”。

登录弹性云服务器。点击“远程登陆”。通过 CloudShell 登录 ecs-hce。
