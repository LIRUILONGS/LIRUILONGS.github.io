DaemonSet зЎ®дҝқжүҖжңүз¬ҰеҗҲжқЎд»¶зҡ„иҠӮзӮ№йғҪиҝҗиЎҢиҜҘ Pod зҡ„дёҖдёӘеүҜжң¬гҖӮ йҖҡеёёпјҢиҝҗиЎҢ Pod зҡ„иҠӮзӮ№з”ұ Kubernetes и°ғеәҰеҷЁйҖүжӢ©гҖӮ дёҚиҝҮпјҢDaemonSet Pods з”ұ DaemonSet жҺ§еҲ¶еҷЁеҲӣе»әе’Ңи°ғеәҰгҖӮиҝҷе°ұеёҰжқҘдәҶд»ҘдёӢй—®йўҳпјҡ
Pod иЎҢдёәзҡ„дёҚдёҖиҮҙжҖ§пјҡжӯЈеёё Pod еңЁиў«еҲӣе»әеҗҺзӯүеҫ…и°ғеәҰж—¶еӨ„дәҺ Pending зҠ¶жҖҒпјҢ DaemonSet Pods еҲӣе»әеҗҺдёҚдјҡеӨ„дәҺ Pending зҠ¶жҖҒдёӢгҖӮ
Pod жҠўеҚ  з”ұй»ҳи®Өи°ғеәҰеҷЁеӨ„зҗҶгҖӮеҗҜз”ЁжҠўеҚ еҗҺпјҢDaemonSet жҺ§еҲ¶еҷЁе°ҶеңЁдёҚиҖғиҷ‘ Pod дјҳе…Ҳзә§е’ҢжҠўеҚ  зҡ„жғ…еҶөдёӢеҲ¶е®ҡи°ғеәҰеҶізӯ–гҖӮиҝҷйҮҢзҡ„й»ҳи®Өи°ғеәҰеҷЁеҚіk8sдёӯи°ғеәҰеҷЁгҖӮ

  • еңЁиө„жәҗеҸ—йҷҗзҡ„е®№еҷЁдёӯиҝҗиЎҢе®ҲжҠӨиҝӣзЁӢиғҪеӨҹеўһеҠ е®ҲжҠӨиҝӣзЁӢе’Ңеә”з”Ёе®№еҷЁзҡ„йҡ”зҰ»жҖ§гҖӮ з„¶иҖҢпјҢиҝҷдёҖзӮ№д№ҹеҸҜд»ҘйҖҡиҝҮеңЁе®№еҷЁдёӯиҝҗиЎҢе®ҲжҠӨиҝӣзЁӢдҪҶеҚҙдёҚеңЁ Pod дёӯиҝҗиЎҢд№ӢжқҘе®һзҺ°гҖӮ дҫӢеҰӮпјҢзӣҙжҺҘеҹәдәҺ Docker еҗҜеҠЁгҖӮ

зӣҙжҺҘеҲӣе»ә Podе№¶жҢҮе®ҡе…¶иҝҗиЎҢеңЁзү№е®ҡзҡ„иҠӮзӮ№дёҠд№ҹжҳҜеҸҜд»Ҙзҡ„гҖӮ з„¶иҖҢпјҢDaemonSet иғҪеӨҹжӣҝжҚўз”ұдәҺд»»дҪ•еҺҹеӣ пјҲдҫӢеҰӮиҠӮзӮ№еӨұиҙҘгҖҒдҫӢиЎҢиҠӮзӮ№з»ҙжҠӨгҖҒеҶ…ж ёеҚҮзә§пјү иҖҢиў«еҲ йҷӨжҲ–з»Ҳжӯўзҡ„ PodгҖӮ з”ұдәҺиҝҷдёӘеҺҹеӣ пјҢдҪ еә”иҜҘдҪҝз”Ё DaemonSet иҖҢдёҚжҳҜеҚ•зӢ¬еҲӣе»ә PodгҖӮ

йқҷжҖҒ Pod

йҖҡиҝҮеңЁдёҖдёӘжҢҮе®ҡзҡ„гҖҒеҸ— kubelet зӣ‘и§Ҷзҡ„зӣ®еҪ•дёӢзј–еҶҷж–Үд»¶жқҘеҲӣе»ә Pod д№ҹжҳҜеҸҜиЎҢзҡ„гҖӮ иҝҷзұ» Pod иў«з§°дёәйқҷжҖҒ PodгҖӮ дёҚеғҸ DaemonSetпјҢйқҷжҖҒ Pod дёҚеҸ— kubectl е’Ңе…¶е®ғ Kubernetes API е®ўжҲ·з«Ҝз®ЎзҗҶгҖӮ йқҷжҖҒ Pod дёҚдҫқиө–дәҺ API жңҚеҠЎеҷЁпјҢиҝҷдҪҝеҫ—е®ғд»¬еңЁеҗҜеҠЁеј•еҜјж–°йӣҶзҫӨзҡ„жғ…еҶөдёӢйқһеёёжңүз”ЁгҖӮ жӯӨеӨ–пјҢйқҷжҖҒ Pod еңЁе°ҶжқҘеҸҜиғҪдјҡиў«еәҹејғгҖӮ

Deployments

DaemonSet дёҺ Deployments йқһеёёзұ»дјјпјҢ е®ғд»¬йғҪиғҪеҲӣе»ә PodпјҢе№¶дё” Pod дёӯзҡ„иҝӣзЁӢйғҪдёҚеёҢжңӣиў«з»ҲжӯўпјҲдҫӢеҰӮпјҢWeb жңҚеҠЎеҷЁгҖҒеӯҳеӮЁжңҚеҠЎеҷЁпјүгҖӮе»әи®®дёәж— зҠ¶жҖҒзҡ„жңҚеҠЎдҪҝз”Ё DeploymentsпјҢжҜ”еҰӮеүҚз«ҜжңҚеҠЎгҖӮ еҜ№иҝҷдәӣжңҚеҠЎиҖҢиЁҖпјҢеҜ№еүҜжң¬зҡ„ж•°йҮҸиҝӣиЎҢжү©зј©е®№гҖҒе№іж»‘еҚҮзә§пјҢжҜ”зІҫзЎ®жҺ§еҲ¶ Pod иҝҗиЎҢеңЁжҹҗдёӘдё»жңәдёҠиҰҒйҮҚиҰҒеҫ—еӨҡгҖӮ еҪ“йңҖиҰҒ Pod еүҜжң¬жҖ»жҳҜиҝҗиЎҢеңЁе…ЁйғЁжҲ–зү№е®ҡдё»жңәдёҠпјҢе№¶дё”еҪ“иҜҘ DaemonSet жҸҗдҫӣдәҶиҠӮзӮ№зә§еҲ«зҡ„еҠҹиғҪпјҲе…Ғи®ёе…¶д»– Pod еңЁиҜҘзү№е®ҡиҠӮзӮ№дёҠжӯЈзЎ®иҝҗиЎҢпјүж—¶пјҢ еә”иҜҘдҪҝз”Ё DaemonSetгҖӮ
